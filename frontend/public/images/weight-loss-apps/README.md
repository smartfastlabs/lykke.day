# Weight Loss Apps Images

This directory contains thumbnail images for the weight loss apps page.

## Required Images

The following images need to be added (recommended size: 600x400px or similar aspect ratio):

- `myfitnesspal.jpg` - MyFitnessPal app
- `noom.jpg` - Noom app
- `loseit.jpg` - Lose It! app
- `weightwatchers.jpg` - WeightWatchers app
- `cronometer.jpg` - Cronometer app

## Image Guidelines

- Format: JPG or PNG
- Recommended dimensions: 600x400px (3:2 aspect ratio)
- File size: Optimized for web (< 200KB each)
- Content: App screenshots, logos, or branded imagery that represents each weight loss app
