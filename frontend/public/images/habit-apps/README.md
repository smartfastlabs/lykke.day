# Habit Apps Images

This directory contains thumbnail images for the habit tracking apps page.

## Required Images

The following images need to be added (recommended size: 600x400px or similar aspect ratio):

- `habitica.jpg` - Habitica app
- `streaks.jpg` - Streaks app
- `productive.jpg` - Productive app
- `done.jpg` - Done app
- `loop-habit-tracker.jpg` - Loop Habit Tracker app
- `way-of-life.jpg` - Way of Life app
- `strides.jpg` - Strides app
- `fabulous.jpg` - Fabulous app

## Image Guidelines

- Format: JPG or PNG
- Recommended dimensions: 600x400px (3:2 aspect ratio)
- File size: Optimized for web (< 200KB each)
- Content: App screenshots, logos, or branded imagery that represents each habit tracking app
