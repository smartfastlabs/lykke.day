# Meditation Apps Images

This directory contains thumbnail images for the meditation apps page.

## Required Images

The following images need to be added (recommended size: 600x400px or similar aspect ratio):

- `headspace.jpg` - Headspace app
- `calm.jpg` - Calm app
- `insight-timer.jpg` - Insight Timer app
- `ten-percent.jpg` - Ten Percent Happier app
- `waking-up.jpg` - Waking Up app
- `balance.jpg` - Balance app
- `oak.jpg` - Oak Meditation app
- `simple-habit.jpg` - Simple Habit app

## Image Guidelines

- Format: JPG or PNG
- Recommended dimensions: 600x400px (3:2 aspect ratio)
- File size: Optimized for web (< 200KB each)
- Content: App screenshots, logos, or branded imagery that represents each meditation app
