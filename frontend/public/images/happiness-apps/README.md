# Happiness Apps Images

This directory contains thumbnail images for the happiness apps page.

## Required Images

The following images need to be added (recommended size: 600x400px or similar aspect ratio):

- `happify.jpg` - Happify app
- `superbetter.jpg` - SuperBetter app
- `gratitude.jpg` - Gratitude app
- `jour.jpg` - Jour: Gratitude Journal app
- `shine.jpg` - Shine: Self-Care app
- `reflectly.jpg` - Reflectly app
- `day-one.jpg` - Day One Journal app
- `presently.jpg` - Presently: Gratitude Journal app

## Image Guidelines

- Format: JPG or PNG
- Recommended dimensions: 600x400px (3:2 aspect ratio)
- File size: Optimized for web (< 200KB each)
- Content: App screenshots, logos, or branded imagery that represents each happiness app
